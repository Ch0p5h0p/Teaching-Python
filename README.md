# Teaching-Python

A series of files for introducing people to Python

## Series
0. Setting up Python (o-python.py)
1. Basics (1-basics.py)
   - Defining variables
   - Variable types (int, str, etc)
   - Recasting variables
   - Defining functions
   - Indentation rules
   - Calling functions
   - Return statements
   - Defining lists
   - Indexing lists
   - List functions (len, push, pop, insert, etc)
   - Defining dictionaries
   - Indexing dictionaries
   - Single- and multi-line comments
   - Input and output, such as print() and input()
2. Libraries (2-libraries.py)
   - Imports
   - Library installation
   - Library data access
4. Control structures (3-control-structures.py)
   - If/elif/else
   - Match/case
5. Loops (4-loops.py)
   - For
   - While
6. Classes (5-classes.py)
   - Definition
   - Constructors & destructors
   - Class methods
   - Built-in methods
   - Casting handlers (`__str__()`, `__int__()`)
   - Instantiation
   - Dataclasses
